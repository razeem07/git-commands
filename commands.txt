>git init [to initialize git]

>git status [to list all untracked files]

>git add -A [to adding all untracked file to staging area]

>git commit -m "commit message"[to commit added file to local repository]

>git clone [to take exact copy from remote repository to local workspace]

>git pull [to take changes from remote repository to local workspace]

>git push [to sent changes from local repository to remote repository]

>git config --list [Configuration listing]

>git config --global user.name "username" [to add username]

>git config --global user.email "email" [to add email]

>git remote add remote_name remote_path

>git push remote_name branch_name
